# 🌱 EcoLar

> **A Python application designed to monitor and analyze residential energy consumption, encouraging sustainable habits through energy usage estimation, cost simulations, and personalized recommendations for efficient electricity consumption.**

---

# 📌 Overview

**EcoLar** is an academic project developed for the **Project 1** course in the **Bachelor's Degree in Information Technology Management (GTI)** at **CESAR School**.

The project was created as part of the **Energy Transition** challenge, with the objective of developing a simple, accessible, and educational solution to help users better understand how their daily habits affect household electricity consumption.

By combining energy consumption calculations, cost estimation, simulations, and educational recommendations, EcoLar aims to promote greater awareness of energy efficiency and encourage more sustainable behavior.

Beyond software development, the project provided practical experience in software engineering, system architecture, project management, teamwork, and problem-solving.

---

# 🎯 Project Objectives

The project aims to develop an application capable of:

* Monitoring residential energy consumption;
* Estimating electricity costs;
* Generating consumption reports;
* Simulating energy-saving scenarios;
* Encouraging sustainable habits through educational recommendations;
* Raising awareness of efficient energy usage.

# 🏗️ System Architecture

EcoLar follows a layered architecture designed to improve maintainability, scalability, and code organization.

```text
Views
│
├── Controllers
│
├── Services
│
├── Repositories
│
└── Utils
```

### Views

Responsible for user interaction through the command-line interface.

### Controllers

Coordinate the application flow and communication between layers.

### Services

Implement business rules and energy consumption calculations.

### Repositories

Handle data storage and retrieval.

### Utils

Provide helper functions, validations, and formatting utilities.
